# Answer: the race was **Bubba Gump Shrimp Co’s Run Forrest Run 5K Fun Run**

The Great America/Santa Clara race that fits the clue was **not** styled in the fetched sources as “Bubble Gum Run 5K” or “Bubble Gum 5K.” The best-supported event-listing name—and the only fetched name tying the race title directly to the Great America/Santa Clara venue—is **“Bubba Gump Shrimp Co’s Run Forrest Run 5K Fun Run”**: a Santa Clara race listing uses that exact title as both the page title and heading, categorizes it under “Races & Competitions,” and gives its address as **“Great America 4701 Great America Pkwy, Santa Clara, CA, 95050”** [1]. In other words, the apparent “bubble gum” clue appears to be a mishearing or memory of **Bubba Gump**.

## Why this is the California Great America 5K, not another “Great America” or another Bubble Gum event

The evidence tying the race name to the California park is tight because the same Santa Clara listing gives the title and venue together: **“Bubba Gump Shrimp Co’s Run Forrest Run 5K Fun Run”** at **“Great America 4701 Great America Pkwy, Santa Clara, CA, 95050”** [1]. That address corresponds to California’s Great America: Six Flags’ official California’s Great America directions page gives **“California’s Great America, 4701 Great America Pkwy, Santa Clara, CA 95054,”** and RCDB likewise identifies **California’s Great America** at **4701 Great America Parkway, Santa Clara, California, United States** [2], [3]. A separate archived event listing at the same venue also shows the naming/address convention: it lists a 2010 5K at **“California’s Great America,”** venue **“Paramount’s Great America,”** at **4701 Great America Parkway, Santa Clara, California**, and repeats the location as **“California’s Great America 4701 Great America Pkwy. Santa Clara, CA 95054”** [4].

The event was a 5K race/run rather than just a park activity. The title itself includes **“5K Fun Run”** [1]. Review text on the same listing describes a runner’s **2007** and **2008** participations, says **“Good turnout = 586 ppl ran in total”** in 2008, and discusses finishing and running the whole thing [1]. Another reviewer calls it a **“5K event,”** says the team **“completed the race,”** and gives course notes: **“The 5K course: Run the entire park, head to the large parking lot, and then run the entire park again to complete 5K,”** with race bibs and race packets picked up at the event and a free same-day Great America ticket after the race [1]. A Yelp-derived MapQuest listing at **4701 Great America Pkwy, Santa Clara, CA 95054** independently has review snippets referring to a **“5K event,”** completing the race, a **“first 5K EVER,”** race packets, racing bibs, and **Great America** [5]. A contemporaneous runner blog post dated **Sunday, September 18, 2005**, titled **“Run Forrest Run 5K,”** also says, **“How can I resist a race in Great America?”**, notes sponsorship by **Bubba Gump Shrimp Company**, and describes running through park walkways and the parking lot, finishing, and placement in the race [6].

## Date/year anchored in the sources

The sources support multiple years/listings for the Santa Clara event:

| Evidence | Date/year supported | What it establishes |

| Course-Trained post titled **“Run Forrest Run 5K”** | **September 18, 2005** | A race in Great America sponsored by Bubba Gump Shrimp Company [6] |

The cleanest date-specific citation is the **September 18, 2005** blog post for **“Run Forrest Run 5K”** at Great America [6]. The cleanest exact title-plus-venue source is the UniLocal listing for **“Bubba Gump Shrimp Co’s Run Forrest Run 5K Fun Run”** at **Great America, 4701 Great America Pkwy, Santa Clara** [1].

## Park-name relationship and “old Great America” context

The Santa Clara park has used several names. RCDB’s California’s Great America entry lists the same park with former names including **Great America**, **Paramount’s Great America**, and **Marriott’s Great America** [3]. A GreatAmericaParks history article says the Santa Clara park opened on **March 20, 1976** as **Marriott’s Great America**, and distinguishes it from the later Gurnee, Illinois park, which opened on **May 29, 1976** [7]. Thus a source that says simply **“Great America”** or **“Paramount’s Great America”** at **4701 Great America Parkway, Santa Clara** is referring to the California’s Great America park, not a separate California venue [3], [4].

The wording “old Great America theme park in California” is therefore consistent with either an older name/historical listing context or the park’s pending-closure context. As of a San José Spotlight fact brief in **November 2025**, California’s Great America was still described as an amusement park in Santa Clara, but was slated to close by **2033** and possibly as early as **2027**, with Six Flags not yet having announced a final closure date [8].

## Disambiguation from Six Flags Great America in Illinois

“Great America” can also mean Six Flags Great America in Gurnee, Illinois. That is not the race venue here. The Illinois park’s official directions page gives **Six Flags Great America, 1 Great America Parkway, Gurnee, IL 60031** [9]. The race listing instead gives **Great America, 4701 Great America Pkwy, Santa Clara, CA 95050**, matching California’s Great America’s Santa Clara address [1], [2].

## Similar names checked and ruled out

I found similarly named events, but they do not answer the California Great America clue:

- A literal **“Bubble Gum 5K Walk/Run Presented by Oral Cancer Cause”** was a first annual OCC chip-timed 5K walk/run held **April 14, 2018** at **Walsingham Park, 12615 102nd Ave, Seminole, FL 33778**—a real literal “Bubble Gum” 5K, but in Florida, not at Great America in Santa Clara [10].
- A New Orleans event, **“Bubba Gump Shrimp Company’s ‘Run Forrest Run’”**, was listed for **April 27, 2008** at **429 Decatur St., New Orleans, Louisiana**, and described as a **“5K for Runners and Walkers”**; despite the similar title, it was not the California Great America event [11].
- MileSplit lists another **“Bubba Gump Shrimp Company’s ‘Run Forrest Run’ 2005”** with 5K results in the French Quarter, New Orleans, again showing the broader Bubba Gump/Run Forrest Run 5K naming pattern but not the Santa Clara venue [12].
- Bubba Gump’s current official page uses **“Run Forrest Run 5k”** / **“RUN FORREST, RUN! 5K”** for a 5K at **Kemah Boardwalk, Kemah, Texas**, not California’s Great America [13].

After searches for literal **“Bubble Gum”**/**“Bubblegum”** combined with Great America, Santa Clara, 4701 Great America Parkway, and older park names, I did not find a source for a California Great America race whose official title literally contains **“Bubble Gum.”** The supported answer is therefore: **Bubba Gump Shrimp Co’s Run Forrest Run 5K Fun Run** [1].
